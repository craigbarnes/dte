@title DTE 1 "November 2017"

@h1 NAME

dte \- A small and flexible text editor

@h1 SYNOPSIS

*dte* [*-hBRV*] [*-c* `command`] [*-t* `tag`] [*-r* `rcfile`] [*-b* `name`] [[+`line`] `file`]...

@h1 OPTIONS

*-c* `command`
	Run `command`, after reading the rc file and opening any `file`
	arguments.

*-t* `tag`
	Go to `tag`. Requires tags file generated by Exuberant Ctags.

*-r* `rcfile`
	Read configuration from `rcfile` instead of $DTE_HOME/rc.

*-b* `name`
	Dump the contents of the built-in rc or syntax file with `name`
	and exit.

*-B*
	Print a list of all built-in config names that can be used with
	the *-b* option and exit.

*-R*
	Don't read the rc file.

*-h*
	Display the help summary and exit.

*-V*
	Display the version number and exit.

@h1 BASIC USAGE

Here's some of the default key bindings. *M-x* means Alt+x and
*^V* is Ctrl+v.

@li *M-s*
Select
@li *M-l*
Select lines
@li *^C*
Copy current line or selection
@li *^X*
Cut current line or selection
@li *^V*
Paste
@li *^Z*
Undo
@li *^Y*
Redo
@li *M-x*
Enter command mode
@li *^F*
Enter search mode
@li *F3*
Search next
@li *F4*
Search previous
@li *^T*
Open new tab
@li *M-*`N`
Activate `N`th tab
@li *^W*
Close tab
@li *^S*
Save file
@li *^Q*
Quit

@h1 COMMAND MODE

Command mode allows you to run various editor commands by using a
language similar to Unix shell. The *next* and *prev* commands switch to
the next/previous file. The *open*, *save* and *quit* commands should be
self-explanatory. For a full list of available commands, see *dterc*(5).

The default key bindings for command mode are:

@li *up*/*down*
Browse previous command history.
@li *tab*
Auto-complete current command or argument
@li *^A*
Go to beginning of command line
@li *^B*
Move left
@li *^C*
Exit command mode
@li *^D*
Delete
@li *^E*
Go to end of command line
@li *^F*
Move right
@li *^K*
Delete to end of command line
@li *^U*
Delete to beginning of command line
@li *^V*
Insert special character
@li *^W*
Erase word
@li *Esc*
Exit command mode

@h1 SEARCH MODE

Search mode allows you to enter an extended regular expression to search
in the current buffer.

The key bindings for search mode are mostly the same as in command mode,
plus these additional keys:

@li *M-c*
Toggle case sensitive search option.
@li *M-r*
Reverse search direction.

@h1 ENVIRONMENT

*DTE_HOME*
	User configuration directory. Defaults to *$HOME/.dte* if not set.

@h1 FILES

*$DTE_HOME/rc*
	Your personal configuration. See *dterc*(5) for available
	commands and options.

*$DTE_HOME/syntax/\**
	Your personal syntax files. These override the syntax files that
	come with the program. See *dte-syntax*(5).

*$DTE_HOME/file-locks*
	Records open files to protect you from accidentally editing files
	opened in another process. Used only if the *lock-files* option is
	enabled.

*$DTE_HOME/file-history*
	Last edited files and cursor positions. Used only if the
	*file-history* option is enabled.

*$DTE_HOME/command-history*
	History of *dterc*(5) commands previously used while in
	command mode.

*$DTE_HOME/search-history*
	History of search patterns previously used in search mode.
